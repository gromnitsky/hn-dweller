To import previously exported settings, just drag-and-drop the .json file in this tab.

Until you hit Save, no options are propagated. This includes the values from the Defaults button. Export exports the current values that may not necessarily be saved yet.

Every list is a string in which entries are separated by newlines. Regexp lists aren't case sensitive, exact match lists are.

The Hostname list has nothing to do with the domain names listed in the brackets on the HN page. To get the actual values:

  1. Press F12 after the HN page is loaded.
  2. Click on the 'Console' tab.
  3. There should be several lines of text. Every line is separated by commas. You need the 2nd field.
